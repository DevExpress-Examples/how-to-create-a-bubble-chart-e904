How to create a Bubble chart

The following example demonstrates how to create a Bubble (http://devexpress.com/Help/Content.aspx?help=XtraCharts&document=CustomDocument5212.htm) chart at runtime. Note that this series view type is associated with the XY-Diagram (http://devexpress.com/Help/Content.aspx?help=XtraCharts&document=CustomDocument5908.htm) type, and you should cast your diagram (http://devexpress.com/Help/Content.aspx?help=XtraCharts&document=CustomDocument6017.htm) object to this type, in order to access its specific options.

You can find sample updates and versions for different programming languages here:
http://www.devexpress.com/example=E904.
